Summary: Latitude vs. X


The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomlish this analysis, we first pulled data from the OpenWeather API to assemble a dataset on over 500 cities.


After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperture, humidity, cloudiness, and wind speed. This site provides the source data and visualizations created as a part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
